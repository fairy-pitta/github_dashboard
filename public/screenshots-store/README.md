# Chrome Web Store Screenshots

These screenshots are resized and formatted for Chrome Web Store submission.

## Specifications

- **Size**: 1280 x 800 pixels
- **Format**: JPEG (24-bit, no alpha channel)
- **Files**: 
  - `screenshot_1_1280x800.jpg`
  - `screenshot_2_1280x800.jpg`
  - `screenshot_3_1280x800.jpg`
  - `screenshot_4_1280x800.jpg`

## Chrome Web Store Requirements

- Maximum 5 screenshots
- Size: 1,280 x 800 or 640 x 400
- Format: JPEG or 24-bit PNG (no alpha channel)
- At least one screenshot is required

These files meet all requirements and are ready for upload.
