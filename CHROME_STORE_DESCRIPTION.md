# GitHub Dashboard - Chrome Web Store Description

## Description

GitHub Dashboard transforms your GitHub.com experience by providing a unified, cross-repository dashboard that aggregates all your GitHub activity in one place. Whether you're working across personal repositories, multiple organizations, or open-source projects, this extension gives you instant visibility into everything that matters.

### Unified Activity Overview

Stop jumping between repositories and organizations. GitHub Dashboard consolidates your entire GitHub activity into a single, beautiful dashboard that appears directly on GitHub.com. See pull requests, issues, and repository updates from all your repositories—personal and organizational—in one unified view.

### Comprehensive Pull Request Management

- Created PRs: Track all pull requests you've created across any repository
- Review Requests: Never miss a PR that needs your review, regardless of which repository it's in
- Reviewed PRs: Keep track of PRs you've already reviewed with clear visual indicators
- Smart Filtering: Filter by review status (approved, changes requested, commented, etc.)
- Review Statistics: See at-a-glance review counts and statuses

### Issue Tracking Across Repositories

View all issues you're involved with across all your repositories and organizations. No more switching between multiple repository pages to check on issues—everything is in one place.

### Repository Management

- All Repositories: Browse recently updated repositories from all sources
- Organization Repos: Dedicated view for organization repositories
- Personal Repos: Quick access to your personal repositories
- Favorite Repositories: Star frequently used repositories for instant access
- Auto-refresh: Automatically loads repositories when switching tabs

### Activity Statistics

Get insights into your GitHub activity with comprehensive statistics:
- Weekly & Monthly Comparisons: See how your activity compares week-over-week and month-over-month
- Commit Tracking: Monitor your commit activity across all repositories
- PR & Review Metrics: Track pull requests created and reviews completed
- Issue Activity: Monitor issues you're involved with
- Comment Activity: See your comment contributions

### Achievements & Streaks

- Contribution Streaks: Track your daily contribution streak
- Achievement Badges: Earn badges for reaching milestones (weekly PRs, monthly commits, reviews, etc.)
- Motivation Messages: Get personalized motivation messages based on your activity

### Beautiful, Customizable Interface

- Multiple Themes: Choose from light, dark, and colorful themes
- Responsive Design: Works perfectly on all screen sizes
- Clean UI: Modern, intuitive interface that doesn't clutter your GitHub experience
- Fast Performance: Smart caching reduces API calls and improves load times

### Secure Authentication

- OAuth Support: Secure OAuth authentication using GitHub's Device Flow (recommended)
- Manual Token Option: Support for Personal Access Tokens if you prefer fine-grained control
- Read-Only Access: The extension only performs read operations—your data is never modified

### Key Benefits

- Save Time: No more switching between multiple repository pages  
- Never Miss Reviews: See all PRs needing your attention in one place  
- Track Progress: Monitor your activity across all repositories  
- Stay Organized: Filter and organize your GitHub activity efficiently  
- Cross-Organization: Works seamlessly across personal and organizational repositories  
- Privacy-Focused: All data stays local, with smart caching for performance  

### Perfect For

- Developers working across multiple repositories
- Team leads managing multiple organization repositories
- Open-source contributors tracking issues and PRs
- Anyone who wants a better overview of their GitHub activity

### How It Works

1. Install the extension
2. Authenticate with GitHub (OAuth recommended)
3. Visit GitHub.com—your personalized dashboard appears automatically
4. Start managing your GitHub activity more efficiently!

Transform your GitHub workflow today. Install GitHub Dashboard and experience a unified view of all your GitHub activity, regardless of repository or organization boundaries.
